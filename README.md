# HR-Analytics-Dashboard
HR Analytics Dashboard – Power BI Project

This project presents an HR Analytics Dashboard built using Power BI to analyze employee attrition and workforce trends within an organization. The dashboard provides key insights into employee demographics, job roles, salary distribution, and factors influencing attrition.

The dashboard highlights important HR metrics such as total employees, attrition count, attrition rate, average age, average salary, and average years at the company. Interactive visualizations help identify patterns in employee turnover across departments, education backgrounds, job roles, salary slabs, and experience levels.

Key insights include:

Attrition analysis by gender, age group, education, and salary slab

Employee attrition across different job roles and departments

Trends based on years at company

Identification of roles and demographics with the highest turnover

This dashboard helps HR teams and decision-makers understand workforce behavior, identify retention risks, and make data-driven HR strategies.
